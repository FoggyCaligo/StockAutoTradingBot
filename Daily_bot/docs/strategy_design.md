# Daily Bot Strategy Design

## 1. 목표

Daily Bot의 목표는 KOSPI200 안에서 장중 단기 반등 또는 미세 호가 우위를 포착해 짧게 진입하고, 빠르게 목표가로 정리하는 것이다.

현재 설계 철학은 다음과 같다.

- 유니버스는 넓게 두고
- 필터는 과도하게 많이 겹치지 않게 유지하며
- 자본 배분은 세션 시작 시점에 고정하고
- 기록은 사후 분석이 가능하도록 충분히 남긴다

## 2. 유니버스

현재 유니버스 구성:

- 소스: `KOSPI200`
- 시가총액 하한: `2500억`
- 거래대금 하한: `30억`
- 추세 필터: OFF

추세 필터는 코드에 남아 있지만 현재 운영값은 `false`다. 최근 운용은 추세보다 실시간 기대수익과 전일 등락 범위를 더 직접적인 조건으로 본다.

## 3. 시그널 계산

각 종목에 대해:

1. 20호가를 조회한다.
2. 호가 균형 기반 기대가격을 계산한다.
3. `sell_tick_offset = 1`을 적용한 목표 매도가를 만든다.
4. 현재가 대비 기대수익률을 계산한다.

핵심 계산:

```text
expect_revenue_percent = (target_sell_price - current_price) / current_price * 100
```

## 4. 현재 최종 필터

현재 실운영 최종 필터는 아래 조건들의 교집합이다.

- `min_expected_return_percent >= 0.3`
- `min_prev_day_change_percent <= -1.0`
- `max_prev_day_change_percent <= 0.0`
- 목표 매도가가 현재가보다 높아야 함

현재 꺼져 있는 필터:

- `max_spread_percent = 0.0`
- `spread_expected_return_multiplier = 0.0`
- `max_intraday_jump_from_prev_scan_percent = 0.0`
- `max_orderbook_ask_depth_ratio = 0.0`

즉, 현 시점의 진입은 “기대수익이 0.3% 이상이면서, 당일 급등주는 아니고, 전일 기준으로는 약세/보합 구간에 있는 종목” 쪽에 더 가깝다.

## 5. 자본 계획

현재 자본 계획은 세션 시작 시점 기준으로 고정한다.

- 최소 슬롯 수: `3`
- 최대 슬롯 수: `10`
- 슬롯 단위: `500만원`
- 동시 신규 매수 상한: `3종목`

장중에 일부 포지션이 정리되더라도 세션 계획은 세션 시작 시점 기준으로 잡는다.

## 6. 청산

기본 청산 흐름:

1. 매수 체결 확인
2. 즉시 목표가 지정가 매도 주문 제출
3. 장중 손절 조건 확인
4. `15:00` 강제 청산
5. `15:15` 브로커 체결 대조

현재 손절 관련 값은 아래와 같지만:

- `stop_loss_percent = 4.5`
- `stop_loss_tick_count = 0`
- `stop_loss_tick_multiplier = 0.0`

실제 장중 손절 로직은 tick 기반 동적 손절이 꺼져 있어 목표가 매도와 장마감 청산의 비중이 더 크다.

## 7. 기록 설계

운영 중 남기는 핵심 기록:

- `market_traces`
- `signals`
- `orders`
- `fills`
- `account_traces`
- `daily_reference_prices`

이 기록으로 다음을 복원할 수 있어야 한다.

- 특정 시점 후보군
- 실제 선택 여부
- 기대수익률 분포
- 전일 등락률 범위
- 강제청산/손절/익절 이유

## 8. 현재 리플레이 해석 한계

중요한 운영상 제약이 하나 있다.

- `2026-06-25` 이전 일부 로그에는 `prev_day_change_percent`가 없다.
- 현재 전략은 이 값을 필터에 직접 사용한다.

따라서 오래된 로그를 사용한 멀티데이 리플레이에서는:

- 현재 라이브 규칙을 완전 동일하게 재현하기 어렵고
- 과거 날짜가 과도하게 탈락할 수 있다

이 문제는 과거 로그 재생성 또는 보정 컬럼 복원 없이는 구조적으로 남아 있다.

## 9. 현재 상태 요약

현재 전략은 공격적으로 많은 종목을 건드리는 구조가 아니다.

- 유니버스는 넓지만
- 기대수익률과 전일등락률 필터가 생각보다 강하고
- 목표가가 현재가보다 높아야 하며
- 결과적으로 장중 최종 후보가 0개 또는 1개 수준으로 줄어드는 경우가 자주 있다

따라서 최근 운용의 핵심 과제는 “후보를 더 넓게 볼 것인가”보다 “현재 필터가 너무 보수적인지, 아니면 그 정도가 맞는지”를 검증하는 쪽이다.
