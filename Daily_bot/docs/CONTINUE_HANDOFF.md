# Daily Bot - CONTINUE HANDOFF

## 프로젝트 개요

이 프로젝트는 주식 시장에서 실시간으로 거래 신호를 분석하고 자동 매매를 수행하는 봇입니다. 주식시장의 수요와 공급의 균형점 이론을 기반으로 미래 가격을 예측하여 매매 결정을 내립니다.

## 핵심 알고리즘: 호가 기반 예상가 도출

### 알고리즘 원리
애덤 스미스의 "수요와 공급의 법칙"을 기반으로 합니다. 호가창의 매수수량(수요)과 매도수량(공급)의 불균형 해소 후의 안정 상태를 찾는 방식입니다.

### 알고리즘 동작 방식
1. **매수호가**(bids)와 **매도호가**(asks)를 1:1로 대응시켜 삭제하며 균형점을 탐색
2. **수요와 공급이 완전히 상쇄**될 때까지 반복
3. **마지막 남은 호가 위치**에서 예상가 도출
4. **균형점의 중간값**을 미래 예상가로 사용

### 세부 알고리즘 구현
```python
def predict_price_from_hoga(snapshot: HogaSnapshot) -> int:
    # 매수호가와 매도호가 복사
    bids = deepcopy(snapshot.bids)
    asks = deepcopy(snapshot.asks)
    
    bid_idx = 0
    ask_idx = 0
    
    # 수요와 공급을 1:1로 상쇄하며 균형점 찾기
    while bid_idx < len(bids) and ask_idx < len(asks):
        offset = min(bids[bid_idx].volume, asks[ask_idx].volume)
        bids[bid_idx].volume -= offset
        asks[ask_idx].volume -= offset
        
        if bids[bid_idx].volume <= 0:
            bid_idx += 1
        if asks[ask_idx].volume <= 0:
            ask_idx += 1
    
    # 마지막 남은 호가 위치에서 예상가 도출
    frontier_bid_price = bids[min(bid_idx, len(bids) - 1)].price
    frontier_ask_price = asks[min(ask_idx, len(asks) - 1)].price
    
    predicted = (frontier_bid_price + frontier_ask_price) // 2
    return round_to_tick(predicted)
```

## 프로젝트 구조

### 핵심 파일
- `orderbook_predictor.py`: 예상가 도출 알고리즘
- `main.py`: 메인 실행 로직
- `broker.py`: 거래소 연동 모듈
- `strategy.py`: 매매 전략 모듈

### 주요 모듈
1. **호가 수집 모듈**: 실시간 호가 정보 수집
2. **예상가 도출 모듈**: 수요공급 균형점 기반 예측
3. **매매 실행 모듈**: 예측 결과에 따른 매매 실행
4. **데이터 저장 모듈**: 거래 데이터 및 성과 기록

## 시장 이론 기반

이 알고리즘은 다음과 같은 시장 이론을 기반으로 합니다:

### 1. 수요공급 법칙
- 가격은 수요와 공급에 의해 결정됨
- 호가창은 수요와 공급의 실시간 정보를 제공함

### 2. 시장 균형 이론
- 수요와 공급이 만나는 지점에서 가격이 결정됨
- 현재의 불균형 상태가 완전히 해소된 미래 시점의 가격을 추정

### 3. 동적 균형점 탐색
- 실시간으로 수요와 공급이 상쇄되는 과정 분석
- 시장 참여자들의 행동을 기반으로 한 예측

## 실행 흐름

1. 실시간 호가 정보 수집
2. 호가 구조 분석 및 균형점 탐색
3. 예상가 도출 (수요공급 균형점의 중간값)
4. 매매 결정 및 실행
5. 거래 데이터 기록 및 성과 분석

## 주요 특징

### 1. 실시간 반응성
- 호가 정보가 실시간으로 업데이트됨
- 시장 변화에 즉각적으로 반응

### 2. 시장 이해 기반
- 수요공급 이론을 기반으로 한 예측 방식
- 실제 거래자들의 행동을 분석

### 3. 동적 예측
- 상황에 따라 예상가는 유동적으로 변화
- 고정된 수식이 아닌, 실시간 정보 기반 예측

## 예상가 도출의 시장 이해

이 알고리즘은 시장의 본질적인 동작 방식을 분석합니다:
- **수요와 공급의 상호작용**을 통해 가격 결정
- **실시간으로 수요공급 불균형을 해소하는 과정**
- **시장 참여자들의 행동을 기반으로 한 예측**
